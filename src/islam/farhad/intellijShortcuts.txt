//CTRL


CTRL + W -> Shrink, CTRL + SHIFT + W -> De Shrink
CTRL + D -> Duplicate
CTRL + E -> Recent Files
CTRL + B -> Field Declaration
CTRL + O -> Override method
CTRL + H -> To view inheritance hierarchy
CTRL + Q -> To check documentation

CTRL + F12 -> Navigate within a file

CTRL + SHIFT  + I -> Preview definition
CTRL + SHIFT  + A -> Find Action
CTRL + SHIFT  + R -> Replace
CTRL + SHIFT  + V -> Select text fragment that you've previously copied
CTRL + SHIFT  + W -> ?
CTRL + SHIFT + ENTER-> Completes a statement
CTRL + SHIFT + SPACE-> use after new keyword or to do type matching

CTRL + ALT + L -> Format line

CTRL + ALT + T -> Wrap a code block
CTRL + ALT + V -> Wrap selected exp into var
